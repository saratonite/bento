# A React Graphql Starter

- React / Nextjs
- Apollo Graphql Server
- Prisma (Data layer)
- Tailwind.css

## Install

#### Clone repo
`bash
    git clone https://github.com/saratonite/bento.git
`

#### Install node dependencies
`bash
npm install 
OR
yarn install
`

#### Run Setup Script
`bash
npm run setup
OR
yarn setup
`

#### Run Development mode
`yarn dev OR npm run dev`


## Build
`yarn build OR npm run build`
